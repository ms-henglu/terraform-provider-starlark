# Starlark Provider Quickstart Examples

This directory contains a set of quickstart examples to help you get started with the Terraform Starlark Provider.

## Examples

*   [01-basic-math](./01-basic-math): Performs simple arithmetic operations.
*   [02-string-processing](./02-string-processing): Demonstrates string manipulation capabilities.
*   [03-complex-logic](./03-complex-logic): Shows how to use functions, conditionals, and loops.
*   [04-external-script](./04-external-script): Illustrates how to load and execute an external Starlark script file.
*   [05-datetime-to-epoch](./05-datetime-to-epoch): Converts an ISO 8601 date string to a Unix timestamp.
*   [06-datetime-from-epoch](./06-datetime-from-epoch): Converts a Unix timestamp to an ISO 8601 date string.
*   [07-parse-cidr](./07-parse-cidr): Parses a CIDR string into network details.
*   [08-deep-merge](./08-deep-merge): Implements a recursive deep merge for maps.
*   [09-while-loop](./09-while-loop): Demonstrates using `while` loops for algorithms like Fibonacci.
